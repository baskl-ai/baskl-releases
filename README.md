# BASKL Releases

Public distribution repository for BASKL AI plugins and models.

## Structure

Downloads are organized as GitHub Releases:

- **Plugins** - Pre-built plugin packages for macOS and Windows
- **Models** - AI model files required by plugins

## Download URLs

Files can be downloaded directly from GitHub Releases:

```
https://github.com/baskl-ai/baskl-releases/releases/download/{tag}/{filename}
```

### For users in China

If GitHub is slow, use a proxy:

```
https://gh-proxy.com/https://github.com/baskl-ai/baskl-releases/releases/download/{tag}/{filename}
```
